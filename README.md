## Dependencies

1) **mkdocs**  
Site: https://github.com/mkdocs/mkdocs/releases
```sh
sudo apt-get install mkdocs
```

1) **mkdocs-material**  
Site: https://github.com/squidfunk/mkdocs-material
```sh
pip install mkdocs-material
```

## Run Locally
```sh
mkdocs serve
```

## Deploy Docs Live
```sh
mkdocs gh-deploy
```
